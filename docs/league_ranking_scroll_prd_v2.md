# 랭킹 페이지 상단 고정 UI 기획 (PRD v2)

## 1. 개요
현재 주간 리그 랭킹 페이지에서 스크롤을 내릴 때 상단의 정보와 최상위 랭커들이 화면 밖으로 사라지는 문제가 있습니다. 사용자 경험을 개선하기 위해, 페이지 타이틀과 최상단 랭커(1위~3위) 아이템을 화면 상단에 고정하여 스크롤 시에도 항상 확인할 수 있도록 변경합니다.

## 2. 요구사항
- **상단 고정 영역**:
  - 뒤로가기 및 새로고침 버튼 영역
  - '주간 리그 랭킹' 타이틀 및 기간 텍스트 영역
  - 1위, 2위, 3위 랭커 아이템
- **스크롤 영역**:
  - 4위 이하 랭커 아이템 및 내 순위(동적 포커스 유지)
- **UI 및 동작 기준**:
  - 모바일 프레임의 배경색(`bg-background`)을 활용하여, 고정된 상단 영역이 투명해지면서 스크롤되는 목록과 겹쳐 보이지 않도록 함.
  - 최상단 고정 영역 아래에 적절한 여백과 그림자(`shadow-sm` 등) 혹은 경계선을 두어, 스크롤되는 콘텐츠와의 구분을 명확히 할 수 있으면 좋음 (단, 시각적 일관성을 해치지 않는 선에서).

## 3. 기대 효과
- 유저가 스크롤을 많이 내리더라도 현재 주차의 정보와 최상위 랭커 시상대를 항상 확인할 수 있어 랭킹 경쟁의 몰입도를 높임.
