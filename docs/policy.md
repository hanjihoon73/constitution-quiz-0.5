### 1. 로그인/로그아웃/계정 관리

1. 사용자는 구글 또는 카카오 계정을 선택해 서비스에 로그인할 수 있음
2. 사용자는 최초 로그인 시 자신의 ‘닉네임’을 설정해야 하며, ‘닉네임’은 다른 사용자와 중복될 수 없음
3. 설정한 ‘닉네임’은 마이페이지 화면에서 수정할 수 있음
4. 마이페이지 화면에서 로그아웃 또는 서비스 탈퇴를 할 수 있음
5. 사용자가 회원에서 탈퇴하면, 사용자의 계정 정보를 즉시 삭제하고, 소셜 계정 연동을 해제하고, 로그인 및 퀴즈 이용 기록은 유지함

### 2. 퀴즈팩

1. 퀴즈팩은 ‘기본 순서(basic_order)’를 가지고 있음 (quizpack_loadmap)
2. 퀴즈팩의 상태는 ‘열림(opened)’, ‘닫힘(closed)’, ‘진행 중(in_progress)’, ‘완료(completed)’의 네 가지 상태만 있음
3. 사용자가 최초 로그인에 성공 후 홈으로 이동하면 해당 사용자의 고유한 퀴즈팩 로드맵이 퀴즈팩의 기본 순서에 따라 생성됨 (user_quizpacks)
4. 홈에서 사용자의 퀴즈팩 로드맵에 따라 퀴즈팩 목록을 보여주고, 1번 퀴즈팩은 열림(opened) 상태가 됨
5. 홈으로 이동할 때마다 사용자가 해야 할 퀴즈팩(current_quizpack)이 자동으로 포커싱 되고 강조 됨
6. 사용자가 해야 할 퀴즈팩(current_quizpack)은 진행 중(in_progress) 또는 열림(opened) 상태의 퀴즈팩임 (진행 중 상태의 퀴즈팩이 우선함)
7. 닫힘(closed) 상태의 퀴즈팩은 시작할 수 없으며, 시작을 시도하면 열리지 않은 퀴즈팩이라고 알려줘야 함
8. 진행 중(in_progress) 상태의 퀴즈팩이 있을 때, 홈에서 다른 상태(열림 또는 완료)의 퀴즈팩을 시작하면 기존에 진행 중이던 퀴즈팩의 기록이 초기화된다고 알려주고 선택권을 줘야 함
9. 사용자는 진행 중(in_progress) 상태의 퀴즈팩과 열림(opened) 상태의 퀴즈팩이 각각 1개씩만 존재할 수 있음
10. 사용자의 퀴즈팩 로드맵 순서를 기준으로, 닫힘(closed) 상태의 퀴즈팩은 자기 자신의 바로 직전 퀴즈팩이 완료(completed) 상태가 되면 열림(opened) 상태로 바뀜
11. 사용자의 퀴즈팩 목록 중 가장 마지막 퀴즈팩을 완료하고 홈으로 이동하면 모든 퀴즈팩의 완료를 축하하는 팝업이 표시됨

### 3. 퀴즈

1. 하나의 퀴즈팩은 다수의 퀴즈를 가질 수 있으며, 사용자가 퀴즈팩에서 풀 수 있는 퀴즈의 최대 개수는 10개임 (추후 어드민에서 개수를 제어할 예정임)
2. 퀴즈는 ‘4지선다(multiple)’, ‘O/X(truefalse)’, ‘빈칸채우기(blankchoice)’의 세 가지 유형만 사용함
3. 퀴즈는 새로 풀 때마다 보기의 순서와 정답이 달라져야 함
4. 모든 퀴즈는 정오답을 선택하고 결과를 확인하기 전에 힌트를 볼 수 있고, 정오답 결과가 나오면 풀이가 자동으로 표시됨
5. 퀴즈를 하나씩 풀 때마다 사용자가 제출한 답의 정오답 처리 결과를 보여주고 저장함
6. 하나의 퀴즈팩을 시작한 후 이미 정오답 결과가 기록된 퀴즈는 앞으로 돌아가서 다시 풀 수 없고 결과만 조회할 수 있음
7. 모든 퀴즈를 다 풀면 퀴즈팩 완료 화면을 보여주고, 여기서 ‘다시풀기’를 선택하거나 ‘결과보기’를 선택해 각 퀴즈별 정오답 결과를 조회할 수 있음
8. 진행 중(in_progress) 상태의 퀴즈팩을 시작하면, 정오답 결과가 기록되지 않은 퀴즈 중 가장 빠른 퀴즈부터 풀어야 하고, 정오답 결과가 기록된 퀴즈는 앞으로 돌아가서 조회할 수 있음
9. 완료한 퀴즈팩을 다시 시작하면 ‘다시풀기’ 또는 ‘결과보기’ 중 선택할 수 있음
9. ‘결과보기’를 선택하면 해당 퀴즈팩을 마지막으로 완료한 시점의 각 퀴즈별 정오답 결과를 1번 퀴즈부터 조회할 수 있음
10. ‘다시풀기’를 선택하면 해당 퀴즈팩의 1번 퀴즈부터 새로 풀 수 있음
