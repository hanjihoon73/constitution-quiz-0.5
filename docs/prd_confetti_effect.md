# PRD: 콘페티(Confetti) 효과 추가

## 1. 개요
사용자에게 성취감을 제공하고 긍정적인 경험을 강화하기 위해 특정 이벤트 발생 시 화면에 콘페티(축하 꽃가루) 효과를 추가합니다.

## 2. 요구 사항

### 2.1. 대상 이벤트 (트리거 조건)
1. **최초 사용자 온보딩 완료 후 웰컴 팝업 표시 직후**
   - 사용자가 처음 가입(온보딩)을 마치고 홈 화면에 진입하여 `WelcomeDialog`가 나타날 때 콘페티 효과를 동작시킵니다.
2. **퀴즈팩(10문제) 완료 화면 렌더링 직후**
   - 퀴즈팩을 모두 풀고 넘어가는 완료 결과 페이지(`QuizCompletePage`)가 로딩을 마치고 화면에 렌더링될 때 콘페티 효과를 동작시킵니다.

### 2.2. 기능 및 디자인 요구사항
- **성능 및 사용성**: 부드럽고 가벼운 애니메이션을 제공하기 위해 `canvas-confetti` 라이브러리를 사용합니다.
- **모듈화**: 콘페티 효과 발생 로직은 커스텀 훅(`useConfetti`)으로 분리하여 다른 곳에서도 쉽게 재사용할 수 있도록 구성합니다.
- **동작 방식**: 화면 중앙 또는 상단에서 자연스럽게 흩뿌려지는 효과를 한 번 실행합니다.

## 3. 기술 스택
- **라이브러리**: `canvas-confetti`, `@types/canvas-confetti`
- **구현 방식**: React Custom Hook (`src/hooks/useConfetti.ts`)

## 4. 제약 사항 및 고려 사항
- 콘페티 라이브러리 설치 시 기존 앱에 성능 저하를 주지 않는 방식이어야 합니다.
- 팝업이 닫힐 때 혹은 언마운트 시 불필요한 메모리 누수가 발생하지 않는 구조여야 합니다. (canvas-confetti 특성상 1회성 함수 호출로 해결 가능)
