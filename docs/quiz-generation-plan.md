## 2. 퀴즈 추출 기준

PRD의 3가지 퀴즈 유형별로 같은 내용도 다르게 출제 가능합니다:

| 유형 | 특징 | 적합한 내용 |
|------|------|------------|
| **4지선다** | 정답 1~2개 선택 | 개념 정의, 조건, 주체, 권한 |
| **OX** | 참/거짓 판단 | 사실 확인, 숫자/기간 검증 |
| **빈칸채우기** | 빈칸 1~3개 | 핵심 용어, 숫자, 고유명사 |

### PRD 목표와의 비교

> PRD: "1,000개 이상의 대한민국 헌법 관련 퀴즈"

**달성 가능합니다.** 단, 다음 전략이 필요합니다:

1. **동일 내용 다중 유형화**: 같은 조문을 4지선다, OX, 빈칸채우기로 각각 출제
2. **세부 항목 분리**: 여러 항이 있는 조문은 항별로 분리 출제
3. **연계 문제 출제**: 관련 조문들을 비교하는 문제 추가
4. **응용 문제 추가**: 판례, 시사 등과 연계한 심화 문제

퀴즈팩 100개(각 10문제) 기준으로 1,000문제를 목표로 하면 충분히 달성 가능한 수치입니다.
